== Description ==

Add Lightbox effect to images on your website.

== Installation ==

1. Download the file and unzip.
2. Upload `custom-lightbox` folder to the `/wp-content/plugins/` folder on your web server
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Gallery Usage =

1. When writing a post, click the "Upload Image" button/icon.
2. If you are inserting a Gallery, you must **ensure** that "Link thumbnails to:" has "Image File" selected. As otherwise it couldn't work.


= Manual Usage =

1. Insert your image thumbnail into your post/page.
2. Wrap the thumbnail in a link to the original/big image.
